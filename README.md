# Auto Screen Brightness NG

Automatically switch the screen brightness depending on the power supply status, modified to work for GNOME 49.

### Installation

The extension is available for installation on the [GNOME Shell Extensions website](https://extensions.gnome.org/extension/9188/auto-screen-brightness-ng/).

To install manually follow the steps below:

- download the latest ZIP version of the extension from GIT, original fork the [releases page](https://github.com/Davidasx/auto-screen-brightness/releases)
- run the following command:

    `$ gnome-extensions install auto-screen-brightness.zip`

- restart your session (logout and login)
- enable & configurate from extensions
